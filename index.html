Multimídia

Multimidia é um conjunto de recursos que tem como objetivo estimular todos os nossos sentidos, mas os mais usados são audição e visão. Tem como significado que a informação digital pode ser representada através de áudio, vídeo e animação em conjunto com mídias tradicionais (texto, gráficos e imagens) simultaneamente.

Tipos de dados multimídia:
    -texto, imagens, áudio e animação;
    - representação e processamento de áudio digital. uso de áudio nas aplicações baseadas em computador. representação digital de som...;
    - tecnologia de vídeo. princípios...;
    - compressão de imagem e vídeo digitais. avaliação de sistemas de compressão...;
    - a música e a multimídia: padrão MIDI.

A Multimídia é uma ferramenta que tem diversas funcionalidades de mídias. Muitos arquivos multimídias podem ser encontrados em acervos, e que necessitam de equipamentos eletrônicos ou digitais para leitura.

Áudio

Para adicionar um áudio ao documento html você precisa adicionar a tag aúdio < audio > que é empregado para incorporar um conteúdo de som ao documento.

Sintaxe:
    < audio src = " arquivo " atributos > < / audio >

A tag áudio pode ser usada com alguns atributos:

  • autoplay - Inicia o áudio automaticamente;
  • controls - Com esse atributo, o navegador mostrará controles de reprodução do áudio, como navegação e volume;
  • loop - Se especificado, o áudio repete automaticamente;
  • muted - Se especificado, o áudio é inicializado silenciado;
  • preload - Usado para carregar arquivos grandes. Possui 3 valores:
    •     -none: O vídeo não será pré-carregado;
        -auto: Carrega o vídeo todo antes da execução;
        -metadata: Carrega apenas metadados do arquivo;
  • poster - Imagem prévia do vídeo, exibida antes de seu carregamento;
  • src - URL do áudio que será incorporado no documento, local ou remoto.

Vídeo

Para adicionar um vídeo < video > ao documento html você precisa adicionar a tag < vídeo > que é empregado para incorporar conteúdo de vídeo em um documento.
    Sintaxe:
    < video src = " arquivo " atributos < / video >

A tag vÍdeo pode ser usada com alguns atributos:, muitos iguais ao do áudio:

  • autoplay - Inicia o vídeo automaticamente assim que possível;
  • controls - Controles de reprodução do vídeo;
  • loop - Se especificado, o vídeo reinicia sua execução ao terminar, automaticamente;
  • width - Largura da área de exibição do vídeo, em pixels;
  • height - Altura da área de exibição do vídeo, em pixels;
  • preload - Usado para carregar arquivos grandes. Possui 3 valores:
    •     -none: O vídeo não será pré-carregado;
        -auto: Carrega o vídeo todo antes da execução;
        -metadata: Carrega apenas metadados do arquivo;
  • poster - Imagem prévia do vídeo, exibida antes de seu carregamento;
  • src - URL do vídeo que será incorporado no documento.

Picture

A tag < picture > é utilizada no htm para que possamos definir mais de uma imagem, também é muito utilizada para apresentar imagens de acordo com as dimensões de cada tela. Dentro desta tag sempre é adicionado a tag < img > e a tag < source >.
Sintaxe:
    < picture >
     ;  < source media=tamanho mínimo/máximo da tela srcset=caminho da imagem>
     ;  < img src=arq.formato alt=descrição da imagem >
    < /picture >




Para especificar vários arquivos de mídia podemos usar a tag < source >, disponibilizando o áudio e vídeo ( ou tag < picture >) em formatos diferentes, para que a os navegadores suportem a reprodução.
Sintaxe:

        < audio | video >
        < source src = " arq.formato1 " type = " audio / video / formato1 " >
        < source src = " arq.formato2 " type = " audio / video / formato2 " >
        < source src = " arq.formato3 " type = " audio / video / formato3 " >
        < audio | video >
